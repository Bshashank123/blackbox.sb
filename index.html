Game Rules
  1. Numbers 1–100 are on the board.
  2. Player 1 must start by selecting an even number.
  3. Next moves = multiply or divide the current number by a prime (result must still be 1–100 and available).
  4. The previously circled number is erased each turn.
  5. If a player cannot move, they lose — the previous player wins!
